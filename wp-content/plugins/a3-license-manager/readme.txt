The a3 License manager plugin is required to activate auto upgrades and support for a3 Premium plugins

== Description ==

When you install an a3 Premium plugin you will be prompted to download and install the a3 License manager plugin

= Features =

* One place to manage all a3 Premium plugins License keys
* Enter License Key and save to activate auto updates for plugins that have an active license subscription

= Localization =

If you do a translation for your site please send it to us and we'll include it in the plugins language folder and credit you here with the translation and a link to your site.

* English (default) - always included.
* a3_license_manager.po file in Language folder for translation.
* [Go here](http://a3rev.com/contact-us-page/) to send your translation files to us.


= Minimum Requirements=

* WordPress 4.0
* PHP version 5.4.0 or greater
* MySQL version 5.5 or greater
